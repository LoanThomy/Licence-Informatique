Qui suis-je :

Je suis un étudiant en première année de licence informatique, ce site est le projet de fin d'année dans la matière Base du Web. J'aime beaucoup faire des sites web, c'est pourquoi j'envisage éventuellement une licence professionnelle (parcours "Développeur web full stack").

Organisation :

Organisation des tâches :

Pour pouvoir m'organiser lors de ce projet, j'ai directement créé un bloc note contenant les choses que je dois faire. Plus précisement une to do list organisé un trois partie (to do, doing, done). Chaque tâches inscrit dans la to do list avait sa propre priorité écrite à la fin : "!!" signifie "très important", "!" signifie "important", "*" signifie : bonus.

Organisation du code :

Afin de rendre le code clair, lisible et compréhensible pour moi et d'autres personnes, j'ai organisé le code javascript de la manière suivante : un retour à la ligne entre les blocs dans une fonction, deux retours à la ligne entre deux fonctions. Ensuite après un retour à la ligne, un commentaire explique qu'est-ce-que le bloc fait.

Par exemple, "//variable qui stock le compteur de favoris sur le DOM" suivi de "const cpt_DOM = document.querySelector("#favorites_counter_p").textContent;"

Problèmes recontrés :

Cookies

J'ai compris que pour faire certaines fonctionnalité dans mon site web, il fallait que j'utilise des cookies pour stocker des informations. Car je voulais que mon JavaScript interagisse avec le DOM de toutes mes pages HTML, j'ai donc abandonné certaines idées. Par exemple, je voulais que le thème du site soit par défaut choisi en fonction du thème par défaut de l'appareil qui se connecte au site. Sauf que dès que l'on clique sur une autre page du site.
